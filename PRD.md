# [서비스 이름] OO 스마트시티 드론 체험 교육 예약 관리 서비스

## [한 줄 소개]

스마트시티 드론 체험 교육 프로그램을 예약하고 관리할 수 있는 웹 서비스로, 사용자는 교육 일정을 조회하고 예약을 신청하며 관리자는 예약 현황을 확인하고 승인, 취소 및 삭제를 수행할 수 있는 서비스

## [핵심 기능 — 5가지]

### 1. 교육 프로그램 소개
교육 일정, 장소, 참가비, 정원 및 교육 내용을 제공한다.

### 2. 교육 예약 신청 (Create)
이용자가 원하는 교육 일정과 시간을 선택하여 예약을 신청한다.

### 3. 예약 현황 조회 (Read)
관리자는 예약자 목록과 신청 인원, 예약 상태를 최신순으로 조회한다.

### 4. 예약 승인 및 정원 관리 (Update)
관리자는 예약 승인·취소를 처리하고, 잔여 정원을 자동으로 관리한다.

### 5. 예약 취소 및 삭제 (Delete)
관리자는 취소되거나 중복된 예약을 삭제하여 예약 정보를 관리한다.
(※ 별도의 "취소" 상태는 두지 않으며, 취소는 예약 삭제로 처리한다.)

## [화면 구성]

### ■ 사용자 페이지 (index.html)

- **상단**: 대표 문구 + 서비스 소개(히어로 영역)
- **교육 프로그램 소개**
  - 교육 목적
  - 교육 일정
  - 교육 장소
  - 참가비
- **교육 안내 카드**
  - 드론 기초 체험
  - 비행 실습
  - 스마트시티 활용 사례
- **교육 예약 폼**
  - 이름
  - 연락처
  - 희망 교육 날짜
  - 신청 인원
  - 요청사항
  - 예약 신청 버튼

### ■ 관리자 페이지 (admin.html)

- **관리자 로그인**
  - 이메일
  - 비밀번호
- **로그인 후**
  - 예약 신청 목록(Table)
    - 예약자 이름
    - 연락처
    - 교육 날짜
    - 신청 인원
    - 예약 상태
  - 기능 버튼
    - [승인]
    - [삭제]
  - 상태 표시
    - 대기(회색)
    - 승인(초록)
  - 로그아웃 버튼

## [권한 설계]

### 참가자 (로그인 없음)
- 교육 프로그램 조회
- 교육 예약 신청

### 관리자 (로그인)
- 예약 목록 조회
- 예약 승인
- 예약 삭제

### 반드시 차단해야 할 것 (RLS 정책 — 보안 핵심)

참가자는 로그인 없이 예약을 신청하므로 `anon` 키에 쓰기 권한이 열린다. 아래를 명시적으로 차단하지 않으면 같은 키로 개인정보 유출·무단 조작이 발생한다.

| 대상(역할) | INSERT | SELECT | UPDATE | DELETE |
|------------|:------:|:------:|:------:|:------:|
| 참가자 (`anon`) | ✅ 예약 신청만 | ❌ 조회 불가 | ❌ | ❌ |
| 관리자 (`authenticated`) | — | ✅ | ✅ | ✅ |

- **개인정보 보호**: 예약 목록(이름·연락처 포함) 조회(SELECT)는 관리자 전용. `anon`의 SELECT는 반드시 차단한다.
- **상태 위조 방지**: 참가자 INSERT 시 `예약 상태`를 직접 지정할 수 없으며, 항상 서버 기본값 "대기"로만 저장된다. 승인은 관리자만 수행한다.
- **관리자 기준**: 참가자 회원가입이 없으므로 Supabase Auth로 로그인한 사용자(`authenticated`)를 관리자로 간주한다.

## [만들지 않는 것 (범위 제외)]

- 참가자 회원가입 기능 없음
- 참가자 로그인 기능 없음
- 예약 수정 기능 없음
- 관리자 회원가입 기능 없음
- 관리자 계정은 Supabase Authentication에서 직접 생성
- 온라인 결제 기능 없음
- 문자 및 이메일 알림 기능 없음
