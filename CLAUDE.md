# CLAUDE.md — 스마트시티 드론 체험 교육 예약 관리 서비스

## 프로젝트 개요
- 스마트시티 드론 체험 교육 예약 관리 서비스 (사용자 페이지 + 관리자 페이지)
- 파일 구성: index.html(사용자), admin.html(관리자) — 각각 단일 파일

## 기술 규칙 (항상 지킬 것)
- HTML/CSS/JavaScript 단일 파일 구성, 프레임워크 사용 금지
- Supabase JS 라이브러리는 CDN 방식으로 불러온다.
- 테이블: reservations
  - name
  - contact
  - hope_date
  - participants
  - request
  - status (기본값: "대기")
  - created_at
- 관리자 인증은 Supabase Authentication을 사용한다.
- service_role Key와 비밀번호는 절대 코드에 작성하지 않는다.
- 모든 데이터(CRUD)는 Supabase를 통해 처리한다.

## 범위 규칙 (절대 만들지 말 것)
- 참가자 회원가입 기능 없음
- 참가자 로그인 기능 없음
- 관리자 회원가입 기능 없음
- 온라인 결제 기능 없음
- 문자 및 이메일 알림 기능 없음
- 예약 수정 기능 없음

## 브랜드 규칙
- 톤: 스마트하고 미래지향적인
- 메인 컬러: #2563EB (Blue)
- 보조 컬러: #0F172A (Dark Navy)
- 포인트 컬러: #22C55E (Green)
- 말투: 존댓말
- 버튼 문구는 행동이 드러나도록 작성
  - "교육 예약하기"
  - "예약 승인"
  - "예약 삭제"
  - "로그인"
  - "로그아웃"

## 코드 스타일
- 주석은 한국어로 작성하며 비전공자도 이해할 수 있도록 설명한다.
- 함수와 변수 이름은 의미가 명확하게 드러나도록 작성한다.
- 중복 코드를 최소화하고 함수 단위로 분리한다.
- HTML, CSS, JavaScript를 읽기 쉽도록 적절히 들여쓰기한다.

## 대화 규칙
- 파일을 수정할 때는 무엇을 왜 수정하는지 한 줄로 먼저 설명한 후 변경(diff) 내용을 보여준다.
- 새로운 파일을 생성할 경우 파일명과 생성 위치를 먼저 안내한다.
- 코드 생성 시 바로 실행 가능한 형태로 작성한다.
- 오류가 발생할 가능성이 있는 부분은 원인과 해결 방법을 함께 설명한다.

## UI/UX 규칙
- 모바일 우선(Responsive)으로 구현한다.
- 전체 레이아웃은 깔끔하고 직관적으로 구성한다.
- 카드 UI와 둥근 모서리(Border Radius)를 사용한다.
- 버튼에는 hover 효과를 적용한다.
- 입력 폼은 유효성 검사를 수행한다.
- 로딩 및 성공/실패 메시지를 사용자에게 표시한다.

## 데이터 처리 규칙
- 예약 신청 완료 후 입력 폼을 초기화한다.
- 예약 신청 성공 시 안내 메시지를 표시한다.
- 관리자 화면은 최신 예약 순으로 정렬한다.
- 예약 상태 변경 후 목록을 자동으로 새로고침한다.
